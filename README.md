# 🌌 Orbit – Entertainment Hub

A modern entertainment platform inspired by Netflix, Spotify, Prime Video, and Steam, built using **HTML, CSS, and JavaScript**.

---

## ✨ Features

- 🎬 Hero section with featured content
- 🔍 Live search with suggestions
- ❤️ Favorite movies
- 🎨 Multiple UI Themes
  - Midnight Blue
  - Purple Neon
  - Gold Luxury
  - Ocean Blue
  - Cyberpunk
- 🎥 Movies
- 📺 Series
- 🍥 Anime
- 🎮 Games
- 🎵 Music
- 🤖 Orbit AI Section
- ✨ Smooth hover animations
- 📱 Responsive modern UI
